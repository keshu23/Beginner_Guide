MODULE_01: HOW LLMS WORK
PHASE_01

Tokenization

LLMs don't see words; they see numbers. Tokenization is the process of breaking down raw text into smaller chunks called "tokens". These can be words, parts of words, or even characters.

// Input: "Hello AI"

PHASE_02

Vector Embeddings

Each token is mapped to a high-dimensional vector—a string of numbers that represents its semantic meaning. Words with similar meanings are placed closer together in this mathematical "vector space".

PHASE_03

Self-Attention

This is the "brain" of the transformer. The model looks at every token in the sequence and decides which ones are most relevant to each other. In "The cat sat on the mat", it knows 'sat' relates to 'cat'.

MODULE_02: AGENTIC AI LOGIC

The Step-by-Step Autonomous Loop

Unlike standard LLMs that just "predict text", Agents "predict actions". They follow a recursive loop to solve complex problems.

1. Perception

The agent receives a goal and observes its environment (API data, user files, previous logs).

2. Reasoning (Planning)

Using patterns like **ReAct** (Reason + Act), the agent thinks about the steps needed. "I need to search the web, then summarize the results."

3. Action (Tool Use)

The agent selects and executes a tool (e.g., Python script, Web Search, Database Query).

4. Observation & Refinement

The agent analyzes the tool's output. If the goal isn't met, it loops back to step 2 with new knowledge.

TASK: STOCK_ANALYSIS_AGENT
> THOUGHT: I need to find the current stock price of Tesla.
> ACTION: google_search("TSLA stock price")
> OBSERVATION: Price is $180.34 (down 2%).
> THOUGHT: I have the data. Now I will write the summary.
> FINAL_ANSWER: Tesla is currently trading at $180.34.
TASK: RESEARCH_AGENT
> THOUGHT: I need to research recent breakthroughs in quantum computing.
> ACTION: arxiv_search("quantum computing 2024")
> OBSERVATION: Found 3 papers on topological qubits.
> THOUGHT: I should summarize the abstract of the first paper.
> FINAL_ANSWER: Recent breakthroughs include stable topological qubits...
TASK: CODE_DEBUGGER_AGENT
> THOUGHT: The user has a SyntaxError in line 45.
> ACTION: analyze_syntax("main.py")

> THOUGHT: I will apply the fix.
> FINAL_ANSWER: Fixed line 45 by adding ")".
